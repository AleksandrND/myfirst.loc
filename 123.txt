1. Что такое системы контроля версий?
Система контроля версий (СКВ) — это система, регистрирующая изменения в одном или нескольких файлах с тем,
чтобы в дальнейшем была возможность вернуться к определённым старым версиям этих файлов.
		********************************
2. В чем разница между гит и гитхаб?
GitHub — веб-сайт для хостинга и совместной разработки проектов.
Git  — распределённая система управления версиями файлов.
		********************************
3. Как в твоем проекте myfirst.loc  используя гит "откатится" на состояние 31 октября 18:36 GMT+4. 
("Откатиться", значит получить все файлы проекта в том состоянии, в котором они были на этот момент.)
31 октября 18:36 GMT+4. - Такого коммита у меня нет))
Откатиться к конкретному коммиту :
git reset --hard (ХЭШ)
Аналогично предыдущему, но файлы на диске остаются без изменений:
git reset --soft (ХЭШ)

5. Для чего нужны ветки гит, как их создать, как переключаться между ветками?
создать новую ветку:
git checkout -b (имя)
Переключение:
git checkout (имя)
